# STYLISHIVE

A minimalist fashion e-commerce landing page built with HTML and Tailwind CSS.

## Overview

STYLISHIVE is a single-page storefront showcasing curated clothing collections. The design uses a clean near-white palette, serif headings, and editorial photography for a premium retail feel.

## Features

- **Hero section** — split layout with headline, CTAs, social proof, and feature cards
- **Product catalog** — filterable grid (All, Men, Women, Kids)
- **Shopping bag** — add, remove, and adjust quantities; cart persists in `localStorage`
- **Search** — filter products by name with category dropdown
- **Category browsing** — visual category cards linked to product filters
- **Promo & collections** — seasonal sale banner and featured lookbook grid
- **Responsive layout** — mobile menu and cart drawer for smaller screens

## Tech Stack

- HTML5
- [Tailwind CSS](https://tailwindcss.com/) (CDN)
- Vanilla JavaScript
- Google Fonts — Inter & Playfair Display
- Images from [Unsplash](https://unsplash.com/)

## Getting Started

No build step or dependencies required.

### Option 1 — Open directly

Double-click `index.html` or open it in any modern browser.

### Option 2 — Local server

```bash
cd "cloths website"
python3 -m http.server 8080
```

Then visit [http://localhost:8080](http://localhost:8080).
